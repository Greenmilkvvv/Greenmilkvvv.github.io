# 西贡的眼睛：南越·中央情报办公室

中央情报办公室（越南语：Phủ Đặc ủy Trung ương Tình báo）是越南共和国（下文称南越）政府的国家战略情报机构，总部设于西贡。

本文是对零星冷战历史资料的简单整理，旨在介绍这一已经被中文互联网遗忘的情报组织的简况。**本文绝对不存在任何来自于AI幻觉的廉价内容。**

------

## 一切的开端
1945年九月，胡志明在河内宣读《独立宣言》，宣告越南民主共和国成立，决心结束法国的殖民统治。然而，法国势力卷土重来，意图重建其印度支那联邦。经过近九年的战争，1954年奠边府战役的炮火，最终为法国的殖民时代画上了句号。同年七月，在华盛顿、莫斯科、伦敦和北京等方的介入和推动下，《日内瓦协议》签订。这份旨在恢复印度支那和平的协议，其核心内容是在越南**以北纬17度线为临时军事分界线**，法军与越盟武装各自向南北集结，并约定**于1956年举行全国普选**，以实现国家统一。然而，统一的愿景很快被冷战的铁幕撕裂。协议墨迹未干，南北便走上了截然不同的道路。在北方，胡志明领导的越南民主共和国（北越）巩固了政权；在南方，获得美国强力支持的吴廷琰，**于1955年宣布废除君主保大政权，成立越南共和国**（RVN），并拒绝举行预定的全国大选。

自此，一个古老的国家在事实上以及法律意义上被一分为二，两个敌对政权在两大阵营的对峙前沿隔线相望，就好像在世界上其他许多地方发生的悲剧一样。分界线的存在，绝不仅是一道地理上的分割线。“这不是结束，甚至不是结束的开始，只是开始的结束。”
接下来要讲述的，是在这片动荡与矛盾的土壤中，南越的一个特殊机构——中央情报办公室（1961-1975）——诞生与运作的故事。
