# 科目二 曲线行驶

> **不限时间 全程尽量慢速** <br>
> **曲线行驶打方向都是一圈**

- 尽量居中直线驶入曲道
- 直行至**车窗左下角与右侧车道黄线距离仅有 1 虎口距离**, **左打一圈**. 
- **保持慢速**, 通过左后视镜, **观察左边黄线同车辆左后轮的距离**, 保持**30 cm**, **修正距离**
- 行驶至**车窗左下角与左侧车道黄线重合 稍过一点**, **回正**.
- 直行至**前方黄线消失在车窗视野之后一点**, **右打一圈**.

> 这边可以探头出窗外

- **保持慢速**
- **探头出窗外**, **观察左前轮**, **不断调整前轮转向, 使其与左侧黄线保持安全距离**
