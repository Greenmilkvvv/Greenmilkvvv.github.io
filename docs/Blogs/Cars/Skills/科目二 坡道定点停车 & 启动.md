# 科目二 坡道定点停车 & 启动

## 停车

- 上坡前, 修正车身, **通过左右微调, 让雨刮器沿着右侧黄线直走**. 
- 当看到**后视镜下沿贴到白实线中间**, **离合踩到底**, **同时右脚紧踩刹车**. 
- **上拉手刹, 等待 3 秒**. 

## 启动

- **向下松手刹**, **一定是依然紧踩刹车, 以防止溜车**. 
- **紧踩住离合**, **慢慢松开离合**, 等待车辆**轻微抖动**, **再抬起一点点**, 以使之**剧烈抖动**, **之后左脚就可以定住不动**.

> 注意: 关于半联动状态, 可以看转速表, 当指针下方微微抖动, 就是半联动了. <br>
> 注意: 离合应该不需要全松开

- **慢慢松开刹车**, **车辆开始启动**.
