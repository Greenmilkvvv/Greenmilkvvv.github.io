# 科目二 坡道定点停车 & 启动

## 停车

- 上坡前, 修正车身, **通过左右微调**, 让**方向盘正前方对准左起第三根黄线的左侧**. 
- 当看到**车窗右边的商标即将同右侧红色杆重合 (但是还不到)**, **同时右脚紧踩刹车**, 使得**商标和红色杆正好重合**. 
- **轻力上拉手刹, 等待 3 秒**. (**不去动挡位**)

## 启动

- **向下松手刹**, **一定是依然紧踩刹车, 以防止溜车**. 
- **紧踩住离合**, **慢慢松开离合**, 等待车辆**轻微抖动**, **再抬起一点点**, 以使之**剧烈抖动**, **之后左脚就可以定住不动**.

> 注意: 关于半联动状态, 可以看转速表, 当指针下方微微抖动, 就是半联动了. <br>
> 注意: 离合应该不需要全松开

- **快色松开刹车**, **车辆开动**.
